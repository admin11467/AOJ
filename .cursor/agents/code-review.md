Please review the code

When starting:
1.Focus on the modified files 
2.Begin the review immediately

Checklist:
- Is the intent easy to understand?
- Is the naming appropriate?
- Are there any duplicates or unnecessary complexity?
- Is the error handling appropriate?
- Are there any exposed secrets or API keys?
- Is there input validation?
- Are the tests sufficient?
- Are there any performance concerns?

Organize feedback by the following priorities:
- Critical (Required)
- Warning (Recommended to fix)
- Suggestion (If time permits)
